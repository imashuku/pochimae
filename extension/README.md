# ポチマエ Chrome拡張

Amazonの商品ページでツールバーのボタンを1クリックすると、販売元情報（出荷元・販売元・セラープロフィールの特定商取引法ブロック）を自動で集め、新しいタブでポチマエの確認結果を開きます。

## 設計（審査・プライバシー対応）

- **Manifest V3／権限は `activeTab` と `scripting` のみ**。ホスト権限なし・常駐コンテンツスクリプトなし。ボタンを押した瞬間だけ、いま見ているタブに収集関数を注入する
- 収集した情報は **URLフラグメント（#以降）** でポチマエに渡す。フラグメントはサーバーに送信されないため、販売元情報・商品URLがサーバーログに残らない
- 拡張自体はデータを一切保存・送信しない（ストレージ未使用・外部通信はAmazonのセラーページ取得（同一オリジン）とポチマエを開くことのみ）
- 収集ロジックは `lib/bookmarklet.ts` と同一（実Amazonページで E2E 検証済み）

## ローカルで試す

1. Chromeで `chrome://extensions` を開く
2. 右上「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」→ この `extension/` フォルダを選択
4. Amazonの商品ページを開き、ツールバーの「ポ」アイコンをクリック

## Chrome Webストア提出手順

1. [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole) でデベロッパー登録（初回のみ・登録料 $5）
2. パッケージ作成:
   ```bash
   cd extension && zip -r ../pochimae-extension-v0.1.0.zip . -x '.*' -x '__MACOSX*'
   ```
3. Dashboard →「新しいアイテム」→ zipをアップロード
4. ストア掲載情報:
   - カテゴリ: ショッピング ／ 言語: 日本語
   - スクリーンショット: 1280x800（Amazonページ＋結果画面の2枚推奨）
   - 説明文はこのREADME冒頭を流用
5. プライバシータブ（審査で最重要）:
   - **単一用途の説明**: 「Amazonの商品ページに表示されている販売元情報を、ユーザーのクリック時にのみ読み取り、確認ツール（pochimae.vercel.app）に引き渡す」
   - **activeTabの理由**: 「ユーザーがボタンを押した時だけ、表示中のAmazon商品ページから販売元情報を読み取るため。常時のホスト権限は要求しない」
   - **scriptingの理由**: 「クリック時に収集スクリプトを1回実行するため」
   - **データ使用**: ユーザーデータの収集・保存・第三者提供なし（Remote code なし）
6. 審査提出（通常数日〜2週間）

## バージョン更新

`manifest.json` の `version` を上げて zip を再アップロード。
